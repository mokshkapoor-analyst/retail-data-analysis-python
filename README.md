# 📁 Dataset Overview

This project analyzes a retail dataset consisting of customer transactions, product hierarchy, and customer information.

The dataset was created by merging multiple datasets containing:

- Customer demographics
- Product category hierarchy
- Transaction details
- Store type information
- Purchase quantities
- Sales amount
- Return transactions

The final merged dataset (`merged_data`) was used for complete analysis and business insight generation.

---

# 🎯 Business Problem & Analysis

The main objective of this project is to understand:

- How customers purchase and return products
- Which product categories drive maximum revenue
- Which store types contribute most to sales
- Customer distribution across cities
- How returns impact overall profitability
- Identification of high-value customers
- Seasonal and monthly sales behavior

The analysis helps retail businesses improve:

- Product strategy
- Inventory planning
- Customer targeting
- Store performance optimization
- Profitability management

---

# 🧹 Data Preparation & Cleaning

Data cleaning and preprocessing were performed using Python.

## ✔ Data Cleaning

- Handled missing values
- Validated numerical and categorical fields
- Checked data types using `.info()`
- Removed inconsistencies in transaction records

## ✔ Data Merging

- Combined customers, transactions, and product hierarchy datasets
- Used appropriate joins to retain valid transaction history
- Created final dataset: `merged_data`

## ✔ Exploratory Data Analysis (EDA)

- Distribution analysis using histograms
- Product category analysis using bar charts
- Outlier detection using boxplots
- Customer purchase behavior analysis
- Monthly sales & returns trend analysis

# 📊 Key Insights & Findings

## 🛍 Product & Store Insights

- Electronics and Clothing categories contribute significantly to overall revenue
- Certain store types dominate both sales quantity and revenue generation
- High-performing categories consistently drive transaction volume

## 👥 Customer Behavior Insights

- A small group of customers contributes a large share of transactions
- High-value customers generate significant revenue
- Customer spending patterns are concentrated among repeat buyers

## 📍 Geographic Insights

- Majority of customers belong to a single city
- Indicates regional concentration dependency
- Highlights expansion opportunities into underpenetrated regions

## 🔁 Returns Analysis

- Return rate (~9%) significantly impacts profitability
- High returns reduce overall net revenue
- Indicates possible issues with product quality or customer satisfaction

## 📅 Time-Based Trends

- Monthly sales show seasonal fluctuations
- Higher spending observed during specific periods
- Revenue trends vary across product categories and store types
